
## Raspberry PI Notes

### Commands

Update Raspbian OS: `sudo apt-get update ; sudo apt full-upgrade`
Raspberry Config: `sudo raspi-config`

### Power Requirements:

- 5.1V/3.0 amps for Raspberry Pi 4
- 5V/2.5 amps for Raspberry Pi 3
